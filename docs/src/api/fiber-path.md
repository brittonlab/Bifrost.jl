# [FiberPath](@id fiber-path)

Fiber assembly: binds a built `PathSpecCached` to a `FiberCrossSection` and reference
temperature, assembles the fiber-level bend and twist generators `K(s)` and `Kω(s)`, and
applies meta-driven perturbations and thermal length scaling through `modify(fiber)`.
