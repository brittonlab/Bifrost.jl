# [PathIntegral](@id path-integral)

Adaptive propagation of the Jones equation `dJ/ds = K(s)J` and the coupled sensitivity
system for `G = ∂ωJ`, with breakpoint-aware interval decomposition, phase-insensitive
error control, and MCM-friendly closed-form matrix exponentials and DGD extraction. The
Theory section derives the underlying generators.
