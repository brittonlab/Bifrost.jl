FiberCS

Transverse step-index fiber geometry: converts material properties into guided index, dispersion, nonlinearity, and the local birefringence response coefficients consumed by the fiber assembly layer.

Bifrost.FiberCS.cutoff_wavelength — Method
cutoff_wavelength(fiber, T_K; λ_min, λ_max, atol, maxiter)

Bisect for the LP11 cutoff wavelength.

MCM compatibility

T_K must be a scalar (not Particles). The bisection's signbit branching is not defined per-particle; call with pmean(T_K_uncertain) when the temperature is uncertain. The return is always Float64.
